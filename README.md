# NComics

Free comics website using Nuxt + Tailwind

## Official Link

https://ncomics.vercel.app

## Technology

- Nuxt 3
- TailwindCSS
- Swiper
- Other libraries

## Features

- Fully responsive
- Save/delete read history
- Download chapters as PDF for offline reading
